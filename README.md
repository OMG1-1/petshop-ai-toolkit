# 宠物店AI营销工具箱

一个为宠物店提供智能营销解决方案的Web应用，帮助宠物店：
- 生成趣味宠物身份证促进分享
- 自动生成社交媒体文案
- 创建裂变营销海报
- 策划营销活动方案

## 功能特点

### 1. 宠物身份证生成器
- 为宠物生成专属身份证
- 随机生成有趣的宠物称号
- 自带二维码引流功能
- 可下载分享

### 2. AI营销文案生成器
- 一键生成朋友圈文案
- 自动生成小红书笔记
- 抖音视频文案推荐
- 支持多种营销场景

### 3. 裂变海报生成器
- 一键生成营销海报
- 自定义海报主题和颜色
- 包含二维码引流
- 可下载打印分享

### 4. 活动策划器
- 自动策划宠物店活动
- 包含预算和收益预测
- 多种活动类型支持
- 完整执行方案

## 技术架构

### 前端技术
- HTML5
- CSS3 (使用现代Flexbox/Grid布局)
- JavaScript (ES6)
- html2canvas (用于图像下载)

### 部署方式
本项目可以直接部署到Vercel或其他静态网站托管平台

## 部署步骤

### 1. 本地运行
```bash

# 进入项目目录

# 本地预览（使用Python）
```

## 使用说明

### 1. 宠物身份证生成器
1. 输入宠物信息（名字、品种、年龄）
2. 输入宠物店信息和优惠券代码
3. 点击"生成宠物身份证"
4. 下载身份证图片分享

### 2. AI营销文案生成器
1. 选择发布平台（朋友圈、小红书、抖音）
2. 选择活动类型
3. 输入目标宠物类型
4. 点击"生成营销文案"

### 3. 裂变海报生成器
1. 输入海报主题和优惠信息
2. 选择海报色调
3. 点击"生成裂变海报"
4. 下载海报图片分享

### 4. 活动策划器
1. 选择活动类型
2. 输入活动日期
3. 点击"生成活动方案"
