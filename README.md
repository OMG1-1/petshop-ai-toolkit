# 宠物店AI营销工具箱

## 🎯 项目概述

宠物店AI营销工具箱是一个综合性的宠物店数字营销解决方案，遵循**美柚App逻辑**，包含：
- **宠物身份证生成器**（情绪价值工具）
- **宠物健康档案管理**（健康管理工具）
- **积分商城系统**（电商转化路径）
- **社交分享功能**（社交传播工具）

## 📋 项目架构

```
宠物店AI营销工具箱
├── 宠物身份证生成器（情绪价值工具）
├── 宠物健康档案管理（美柚式健康工具）
├── 积分商城系统（电商转化路径）
├── 社交分享功能（社交传播工具）
└── 店主数据分析后台（业务决策工具）
```

## 📊 技术规格

### 技术栈
- **前端**: HTML5 + CSS3 + JavaScript（纯前端解决方案）
- **后端**: 未来扩展为Node.js + Express + PostgreSQL
- **部署**: Netlify静态托管（当前）+ 未来扩展到云服务
- **存储**: 浏览器localStorage（当前）+ 未来扩展到云数据库

### 核心功能
1. **宠物身份证生成器**
   - 照片上传功能
   - 个性化称号生成
   - 二维码引流设计
   - 分享文案生成

2. **宠物健康档案**
   - 体重记录管理
   - 疫苗记录跟踪
   - 体检提醒功能
   - 健康数据统计

3. **积分商城系统**
   - 积分计算机制
   - 商品兑换逻辑
   - 用户积分管理
   - 积分历史记录

4. **社交分享功能**
   - 分享文案优化
   - 社交平台适配
   - 分享积分奖励
   - 分享数据分析
